# [Molstar](https://molstar.org/) [Metaframe](https://docs.metapage.io/docs/what-is-a-metaframe) 


https://molstar.mtfm.io/#?pdbId=7IF5

Standalone embeddable (as iframe) [molstar](https://molstar.org) molecule viewer

Example embedded: https://metapage.io/m/6307f6f4ceca41789ab5bfa4fc0b8b95

### Options reference

https://github.com/molstar/molstar/blob/master/src/apps/viewer/app.ts

## Usage

### Embed in your own html/code:

```html
  <iframe src="https://molstar.mtfm.io/#?pdbId=7IF5" style="overflow: clip; width: 100%; height: 100%;"></iframe>
```

Just change the `pdbId` to your pdb id
